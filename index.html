What I've Learnt So Far About HTML

HTML means Hyper Text Mark Up Language. It describes the structure of a Web page.
There are 3 main building blocks in HTML:
1. Tags.
2. Attributes.
3. Elements.
HTML contains several elements for defining text with a special meaning. E.g, This is a bold text, This is a deleted text, This is an emphasized text etc.
The above sentences are in a paragraph (p) tag. And lastly the the headings (h) tag can be (h1-h6), h1 being the most important heading and h6 being the least important heading.
